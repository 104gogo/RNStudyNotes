D1:React Native 读取本地的json文件 (2016-8-18)
------  
自 React Native 0.4.3，你可以导入的形式，来读取本地的json文件，导入的文件可以作为一个js对象使用。      

### 第一步：导入json文件   

```JavaScript
var customData = require('./customData.json');
```

ES6/ES2015     

```JavaScript
import customData from './customData.json';
```
